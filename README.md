# esp32-doom-raycaster
ESP32 DOOM-style raycaster built in pure C using ESP-IDF. Runs on SH1106 128x64 OLED at ~30fps with keypad controls and LCD HUD

A DOOM-style raycaster built in pure C using ESP-IDF v5.5.3.
Runs on an ESP32 DevKit with an SH1106 128x64 OLED display at ~30fps.

Features:
- Raycasting 3D engine
- Keypad-driven player movement
- Collision detection
- LCD HUD (in progress)

Built as a Grade 12 Electronics project.
